# Swell Container

[![npm version](https://badge.fury.io/js/%40swell-enterprise%2Fcontainer.svg)](https://badge.fury.io/js/%40swell-enterprise%2Fcontainer)
[![License: Apache-2.0](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

>  ⚠ This package is in **active development** and should not be used in production yet. Expect breaking changes until v1.0.0.

> > A lightweight, strictly-typed Object-Oriented Dependency Injection container for TypeScript.

This package is the foundational DI layer of the **Swell Enterprise Ecosystem**, designed to help teams implement robust Domain-Driven Design (DDD) and CQRS patterns in TypeScript without heavy framework lock-in.

---

## Installation

```bash
pnpm install @swell-enterprise/container
```
