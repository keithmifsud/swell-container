# Swell Container

[![npm version](https://badge.fury.io/js/%40swell%2Fcontainer.svg)](https://badge.fury.io/js/%40swell%2Fcontainer)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

>  ⚠ This package is in **active development** and should not be used in production yet. Expect breaking changes until v1.0.0.

> > A lightweight, strictly-typed Object-Oriented Dependency Injection container for TypeScript.

This package is the foundational DI layer of the **Swell Architecture Ecosystem**, designed to help teams implement robust Domain-Driven Design (DDD) and CQRS patterns in TypeScript without heavy framework lock-in.

---

## Installation

```bash
pnpm install @swell/container
```
